#Digital Thermometer with LCD
Intern ID:CITS3610
Full Name: Sanjay S 
Project Name:
Digital Thermometer with LCD
Project Scope:
The Digital Thermometer with LCD is an Arduino-based project that measures the surrounding temperature using an LM35 temperature sensor and displays the temperature in real time on a 16×2 LCD display. This project demonstrates temperature sensing, analog-to-digital conversion, and LCD interfacing, making it suitable for basic embedded systems and monitoring applications.
